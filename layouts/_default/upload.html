Contribute to ChickyBase

Help expand the chicken gene expression database by submitting new images or improving existing data.

Submit New Images

Have in situ hybridization images of chicken embryo gene expression? Submit them via GitHub Issues.

What we're looking for:

  • In situ hybridization (ISH) images showing gene expression patterns
  • Clear embryonic stage information (HH stage preferred)
  • Gene symbol (official NCBI gene symbol if possible)
  • Citation/source information (PubMed ID, DOI, or paper reference)

How to submit:

  1. 1 Go to our GitHub Issues page
  2. 2 Create a new issue with title: [Image Submission] GENE_SYMBOL
  3. 3 Drag and drop your images into the issue (GitHub supports up to 25MB per file)
  4. 4 Include gene symbol, embryonic stage(s), and source/citation in the description
Submit Images on GitHub

Edit Gene Data

Found an error or want to add information to a gene page? Submit a Pull Request on GitHub.

What you can contribute:

  • Fix typos or incorrect information
  • Add or correct gene descriptions
  • Update expression pattern descriptions
  • Add missing citations or references

How to edit:

  1. 1 Find the gene page you want to edit on ChickyBase
  2. 2 Go to our GitHub repo and find the corresponding .md file
  3. 3 Click the pencil icon to edit, make your changes
  4. 4 Submit a Pull Request with a description of your changes
